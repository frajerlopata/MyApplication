# Rejestracja na wydarzenie

Aplikacja Android wykonana w Java.

## Funkcjonalności

- EditText (imię)
- Spinner (typ biletu)
- CheckBox (posiłek)
- Walidacja danych
- Przekazywanie danych przez Intent.putExtra()
- Odczyt danych przez getIntent().getExtra()
- Potwierdzenie lub anulowanie rejestracji
- Odbiór wyniku przez ActivityResultLauncher
